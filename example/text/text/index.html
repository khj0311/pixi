PIXI Text
Example Code

;(function() {
    'use strict';

    window.pixi = window.pixi || {};
    window.pixi.component = window.pixi.component || {};

    pixi.component.text = (function() {
        var obj = {};

        var init = function() {
            obj.section = document.querySelector('.pixi-text');

            if (!!obj.section) {
                setElements();
                setRichText();
                setSkewText();
            }
        };

        var setElements = function() {
            obj.app = new PIXI.Application({
                view: document.getElementById('text'),
                backgroundColor: 0xFFFFFF
            });

            obj.basicText = new PIXI.Text('Basic text in pixi');
            obj.basicText.x = 50;
            obj.basicText.y = 100;

            obj.app.stage.addChild(obj.basicText);
        };

        var setRichText = function() {
            var style = new PIXI.TextStyle({
                fontFamily: 'Arial',
                fontSize: 36,
                fontStyle: 'italic',
                fontWeight: 'bold',
                fill: ['#ffffff', '#00ff99'],
                stroke: '#4a1850',
                strokeThickness: 5,
                dropShadow: true,
                dropShadowColor: '#000000',
                dropShadowBlur: 4,
                dropShadowAngle: Math.PI / 6,
                dropShadowDistance: 6,
                wordWrap: true,
                wordWrapWidth: 440,
                lineJoin: 'round',
            });

            var richText = new PIXI.Text('Rich text with a lot of options and across multiple lines', style);
            richText.x = 50;
            richText.y = 220;

            obj.app.stage.addChild(richText);
        };

        var setSkewText = function() {
            var skewStyle = new PIXI.TextStyle({
                fontFamily: 'Arial',
                dropShadow: true,
                dropShadowAlpha: 0.8,
                dropShadowAngle: 2.1,
                dropShadowBlur: 4,
                dropShadowColor: '0x111111',
                dropShadowDistance: 10,
                fill: ['#ffffff'],
                stroke: '#004620',
                fontSize: 60,
                fontWeight: 'lighter',
                lineJoin: 'round',
                strokeThickness: 12,
            });

            var skewText = new PIXI.Text('SKEW IS COOL', skewStyle);
            skewText.skew.set(0.65, -0.3);
            skewText.anchor.set(0.5, 0.5);
            skewText.x = 300;
            skewText.y = 480;

            obj.app.stage.addChild(skewText);
        };

        return {
            init: init
        }
    })();

    pixi.component.text.init();
})();

🔗 PIXI.Text
https://pixijs.download/release/docs/PIXI.Text.html
Description
Text 개체는 한 줄 또는 여러 줄의 Text를 만들며, CanvasAPI를 사용하여 생성된다.
BitmapText에 비해 이 클래스의 주요 이점은 런타임에 변경할 수 있는 Text 스타일을 크게 제어할 수 있다는 것이다.
주요 단점은 각 Text 조각에 더 많은 메모리를 사용할 수 있는 고유한 Texture가 있다는 것이다. Text가 변경되면 이 Texture를 다시 생성하고 GPU에 다시 업로드해야 하므로 시간이 걸린다.
줄을 분할하려면 Text 문자열에 '\n'을 사용하거나 스타일 개체에서 wordWrap 속성을 true로 변경하고 wordWrapWidth 속성에 값을 지정할 수 있다.

Example

var text = new PIXI.Text('This is a PixiJS text', {
    fontFamily : 'Arial',
    fontSize: 24,
    fill : 0xff1010,
    align : 'center'
});

Properties
new PIXI.Text (text, style, canvas)
이름 타입 설명
text string Text 개체에 표시할 문자열
style object | PIXI.TextStyle 스타일 매개변수
canvas HTMLCanvasElement 텍스트를 그리기 위한 캔버스 요소

Methods
  • destroy (options) : void

    이 텍스트 객체를 파괴한다.

    Sprite와 달리 Text 개체는 대부분의 경우 텍스처가 다른 Sprite와 공유되지 않기 때문에 baseTexture와 텍스처를 자동으로 파괴한다.

이름 타입 기본값 설명
options IDestroyOptions | boolean 옵션 매개변수. true 일 경우 모든 옵션이 해당 값으로 설정된 것처럼 작동
options.children boolean false true일 경우 모든 자식의 destroy 메소드를 호출
options.texture boolean true Sprite의 Texture도 파괴
options.baseTexture boolean true Sprite의 기본 Texture도 파괴
  • getLocalBounds (rect) : PIXI.Rectangle

    Text 개체의 로컬 경계를 가져온다.

이름 타입 설명
rect PIXI.Rectangle 경계를 가져올 직사각형 개체
  • updateText (respectDirty) : void

    Canvas에 Text를 렌더링하고 Texture를 업데이트한다.

    기본적으로 이것은 렌더링 전에 Texture가 올바른지 확인하기 위해 내부적으로 사용되지만, 예외적으로 이 클래스에서 Texture를 "사전 생성"하고 여러 Sprite에서 공유하기 위해 외부에서 호출할 수 있다.

이름 타입 설명
respectDirty boolean Text가 더러워지지 않아 함수가 호출된 경우 Text 갱신을 중지할지 여부를 지정한다.
🔗 PIXI.TextStyle
https://pixijs.download/release/docs/PIXI.TextStyle.html
Description
TextStyle 개체에는 Text 개체를 장식하기 위한 정보가 포함되어 있다.
인스턴스는 여러 Text 객체 간에 공유될 수 있다. 그런 다음 스타일을 변경하면 스타일을 사용하는 모든 Text 개체가 업데이트된다.

Properties
new PIXI.TextStyle (style)
이름 타입 기본값 설명
style object 스타일 매개변수
style.align string 'left' 여러 줄 Text의 정렬('왼쪽', '중앙' 또는 '오른쪽')은 한 줄 Text에 영향을 주지 않는다.
style.breakWords boolean false 줄을 단어 내에서 줄 바꿈할 수 있는지 여부를 나타낸다. wordWrap을 true로 설정해야 한다.
style.dropShadow boolean false Text의 그림자 설정
style.dropShadowAlpha number 1 그림자에 대한 Alpha 설정
style.dropShadowAngle number Math.PI/6 그림자 각도 설정
style.dropShadowBlur number 0 그림자 흐림 반경 설정
style.dropShadowColor string | number 'black' 그림자에 사용할 채우기 스타일
style.dropShadowDistance number 5 그림자의 반경 설정
style.fill string | Array | number | Array | CanvasGradient | CanvasPattern 'black' Text에 사용될 Canvas 채우기 스타일
style.fillGradientType number PIXI.TEXT_GRADIENT.LINEAR_VERTICAL 채우기가 Gradient를 만들기 위한 색상 배열인 경우 Gradient의 유형/방향을 변경할 수 있다.
style.fillGradientStops Array 채우기가 Gradient를 생성하기 위한 색상 배열인 경우 이 배열은 색상의 정지점(0과 1 사이의 숫자)을 설정하여 균일한 간격의 기본 동작을 재정의할 수 있다.
style.fontFamily string | Array 'Arial' 글꼴 모음
style.fontSize number | string 26 글꼴 크기
style.fontStyle string 'normal' 글꼴 스타일
style.fontVariant string 'normal' 글꼴 변형
style.fontWeight string 'normal' 글꼴 두께
style.leading number 0 줄 사이의 공간
style.letterSpacing number 0 글자 사이의 간격
style.lineHeight number 줄 높이
style.lineJoin string 'miter' lineJoin 속성은 생성된 모서리 유형을 설정하며 스파이크 텍스트 문제를 해결할 수 있다.
"miter"(날카로운 모서리 생성), "round"(둥근 모서리 생성), "bevel"(각진 모서리 생성)
style.miterLimit number 10 렌더링된 텍스트의 뾰족함을 줄이거나 늘릴 수 있다.
style.padding number 0 일부 글꼴이 잘릴경우 추가하는 Padding
style.stroke string | number 'black' Text 획에 사용할 Canvas 채우기 스타일
style.strokeThickness number 0 획의 두께
style.trim boolean false 투명 테두리 자르기
style.textBaseline string 'alphabetic' 렌더링되는 텍스트의 기준선
style.whiteSpace string 'pre' 줄 바꿈 및 공백이 축소되거나 보존되는지 여부를 결정한다.
style.wordWrap boolean false 줄 바꿈을 사용해야 하는지 여부
style.wordWrapWidth number 100 텍스트가 줄 바꿈되는 너비

  • clone () : PIXI.TextStyle

    모든 속성을 TextStyle.prototype._default에 지정된 기본값으로 재설정한다.

    TextMetrics.measureFont()에 사용할 글꼴 스타일 문자열을 생성한다.

이름 설명
string TextMetrics.measureFont()에 전달하기 위한 글꼴 스타일 문자열
