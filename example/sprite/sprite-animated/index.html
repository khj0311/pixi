PIXI Sprite - Animated Sprite - Explosion
Example Code

;(function() {
    'use strict';

    window.pixi = window.pixi || {};
    window.pixi.component = window.pixi.component || {};

    pixi.component.spriteAnimated = (function() {
        var obj = {};

        var init = function() {
            obj.section = document.querySelector('.pixi-sprite-animated-explosion');

            if (!!obj.section) {
                setElements();
                setLoader();
            }
        };

        var setElements = function() {
            obj.app = new PIXI.Application({
                view: document.getElementById('spriteAnimated')
            });
            obj.app.stop();
        };

        var setLoader = function() {
            obj.app.loader
                .add('spritesheet', '/content/assets/explosion.json')
                .load(onAssetsLoadded);
        };

        var onAssetsLoadded = function() {
            var explosionTextures = [];
            var i = 0;

            for (i = 0; i < 26; i++) {
                var texture = PIXI.Texture.from(`Explosion_Sequence_A_${i + 1}.png`);
                explosionTextures.push(texture);
            }

            for (i = 0; i < 50; i++) {
                var explosion = new PIXI.AnimatedSprite(explosionTextures);

                explosion.x = Math.random() * obj.app.screen.width;
                explosion.y = Math.random() * obj.app.screen.height;
                explosion.anchor.set(0.5);
                explosion.rotation = Math.random() * Math.PI;
                explosion.scale.set(0.75 + Math.random() * 0.5);
                explosion.gotoAndPlay(Math.random() * 27);
                obj.app.stage.addChild(explosion);
            }

            obj.app.start();
        };

        return {
            init: init
        }
    })();

    pixi.component.spriteAnimated.init();
})();

🔗 PIXI.AnimatedSprite
https://pixijs.download/release/docs/PIXI.AnimatedSprite.html
Description
AnimatedSprite는 Texture 목록으로 묘사된 애니메이션을 표시하는 간단한 방법이다.

Example

var textureArray = [];
var alienImages = [
    'image_sequence_01.png',
    'image_sequence_02.png',
    'image_sequence_03.png',
    'image_sequence_04.png'
];

for (var i = 0; i < 4; i++) {
    var texture = PIXI.Texture.from(alienImages[i]);
    textureArray.push(texture);
}

var animatedSprite = new PIXI.AnimatedSprite(textureArray);
애니메이션 Sprite를 생성하는 더 효율적이고 간단한 방법은 애니메이션 정의가 포함된 PIXI.Spritesheet를 사용하는 것이다.

PIXI.Loader.shared.add('assets/spritesheet.json').load(setup);

function setup() {
var sheet = PIXI.Loader.shared.resources['assets/spritesheet.json'].spritesheet;
animatedSprite = new PIXI.AnimatedSprite(sheet.animations['image_sequence']);
...
}

Properties
new PIXI.AnimatedSprite (textures, autoUpdate)
이름 타입 기본값 설명
textures PIXI.Texture[] | PIXI.AnimatedSprite.FrameObject[] 애니메이션을 구성하는 PIXI.Texture 또는 프레임 개체의 배열
autoUpdate boolean true PIXI.Ticker.shared를 사용하여 애니메이션 시간을 자동 업데이트할지 여부.

Methods
  • PIXI.AnimatedSprite.fromFrames (frames) : PIXI.AnimatedSprite

    프레임 ID 배열에서 AnimatedSprite를 만드는 간단한 방법

이름 타입 설명
frames string[] AnimatedSprite가 텍스처 프레임으로 사용할 프레임 ID의 배열
  • PIXI.AnimatedSprite.fromImages (images) : PIXI.AnimatedSprite

    이미지 ID 배열에서 AnimatedSprite를 만드는 간단한 방법

이름 타입 설명
images string[] AnimatedSprite가 텍스처 프레임으로 사용할 이미지 URL의 배열
  • destroy (options) : void

    AnimatedSprite를 중지하고 파괴한다.

이름 타입 기본값 설명
options object | boolean 옵션 매개변수. true 일 경우 모든 옵션이 해당 값으로 설정된 것처럼 작동
options.children boolean false true일 경우 모든 자식의 destroy 메소드를 호출
options.texture boolean false Sprite의 Texture도 파괴
options.baseTexture boolean false Sprite의 기본 Texture도 파괴
  • gotoAndPlay (frameNumber) : void

    특정 프레임으로 이동하여 AnimatedSprite 재생을 시작한다.

  • gotoAndStop (frameNumber) : void

    AnimatedSprite를 중지하고 특정 프레임으로 이동한다.

이름 타입 설명
frameNumber number 시작하거나 중지할 프레임 인덱스
  • play () : void

    AnimatedSprite를 재생한다.

  • stop () : void

    AnimatedSprite를 중지한다.

  • update (deltaTime) : void

    렌더링을 위해 개체 변환을 업데이트한다.

이름 타입 설명
deltaTime number 마지막 틱 이후의 시간
