PIXI Sprite - Tiling Sprite
Example Code

;(function() {
    'use strict';

    window.pixi = window.pixi || {};
    window.pixi.component = window.pixi.component || {};

    pixi.component.spriteTiling = (function() {
        var obj = {};

        var init = function() {
            obj.section = document.querySelector('.pixi-sprite-tiling-sprite');

            if (!!obj.section) {
                setElements();
                setTicker();
            }
        };

        var setElements = function() {
            obj.count = 0;
            obj.app = new PIXI.Application({
                view: document.getElementById('spriteTiling'),
                backgroundColor: 0x1099bb
            });

            obj.texture = PIXI.Texture.from('/content/images/tile.jpg');
            obj.tilingSprite = new PIXI.TilingSprite(
                obj.texture,
                obj.app.screen.width,
                obj.app.screen.height
            );

            obj.app.stage.addChild(obj.tilingSprite);
        };

        var setTicker = function() {
            obj.app.ticker.add(function(delta) {
                obj.count += 0.005;

                obj.tilingSprite.tileScale.x = 2 + Math.sin(obj.count);
                obj.tilingSprite.tileScale.y = 2 + Math.cos(obj.count);

                obj.tilingSprite.tilePosition.x += 1;
                obj.tilingSprite.tilePosition.y += 1;
            });
        };

        return {
            init: init
        }
    })();

    pixi.component.spriteTiling.init();
})();

🔗 PIXI.TilingSprite
https://pixijs.download/release/docs/PIXI.TilingSprite.html
Description
TilingSprite는 타일링 이미지를 렌더링하는 빠른 방법이다.

Properties
new PIXI.TilingSprite (texture, width, height)
이름 타입 기본값 설명
texture PIXI.Texture TilingSprite의 텍스쳐
width 100 TilingSprite의 너비
height 100 TilingSprite의 높이

Methods
  • PIXI.TilingSprite.from (source, options) : PIXI.TilingSprite

    제공한 소스를 기반으로 새로운 TilingSprite를 생성하는 도우미 함수. 소스는 프레임 ID, 이미지 URL, 비디오 URL, 캔버스 요소, 비디오 요소, 기본 텍스처일 수 있다.

이름 타입 설명
source string | PIXI.Texture | HTMLCanvasElement | HTMLVideoElement 텍스처를 생성할 소스
options Object
options.width number TilingSprite에 필요한 너비
options.height number TilingSprite에 필요한 높이
  • containsPoint (point) : boolean

    이 TilingSprite 내부에 점이 있는지 확인한다.

이름 타입 설명
point IPointData 확인해야 할 Point
  • destroy (options) : void

    이 스프라이트와 선택적으로 텍스처와 자식을 파괴한다.

이름 타입 기본값 설명
options object | boolean 옵션 매개변수. true 일 경우 모든 옵션이 해당 값으로 설정된 것처럼 작동
options.children boolean false true일 경우 모든 자식의 destroy 메소드를 호출
options.texture boolean false Sprite의 Texture도 파괴
options.baseTexture boolean false Sprite의 기본 Texture도 파괴
  • getLocalBounds (rect) : PIXI.Rectangle

    Sprite 개체의 로컬 경계를 가져온다.

이름 타입 설명
rect PIXI.Rectangle 선택적 출력 사각형
